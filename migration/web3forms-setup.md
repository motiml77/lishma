# התראות מייל למנהל בכל הזמנה — Web3Forms (ללא שרת)

החנות שולחת בכל הזמנה חדשה מייל למנהל דרך **Web3Forms** — שירות חינמי שעובד ישירות מהדפדפן, בלי שרת ובלי Gmail API.

## הגדרה (דקה אחת)

1. היכנס ל-[web3forms.com](https://web3forms.com).
2. הזן את **כתובת המייל** שאליה תרצה לקבל את ההזמנות → לחץ **Create Access Key**.
3. תקבל במייל **Access Key** (מחרוזת ארוכה). העתק אותה.
4. באתר החנות → היכנס כמנהל → פתח את ניהול ההזמנות.
5. לחץ על הכפתור **🔑 מפתח שליחת מייל** → הדבק את ה-Access Key → שמור.
6. (אופציונלי) לחץ **מייל מנהל** והזן את אותו מייל — כדי שהתשובות יחזרו אליו.

מעכשיו — בכל הזמנה חדשה יישלח מייל אוטומטי עם כל פרטי ההזמנה (שם הספר, פרטי הלקוח, כמות, נקודת איסוף, סה"כ ועוד).

## איך זה עובד בקוד
- ה-Access Key נשמר ב-Firestore: `settings/notifications.web3formsKey` (ניתן לעריכה רק מהניהול).
- בעת שליחת הזמנה, **בנוסף** לשמירה ב-Firestore, נשלחת בקשת POST ל-`https://api.web3forms.com/submit`.
- אם השליחה נכשלת — ההזמנה עדיין נשמרת בהצלחה; השגיאה נרשמת רק בקונסול.

## אבטחה
Access Key של Web3Forms מיועד לשימוש בצד-לקוח (frontend) — זה האופן שבו השירות בנוי. אין צורך בשרת.
אם בעתיד תרצה להסתיר אותו — אפשר להעביר את הקריאה ל-Cloud Function, אבל זה לא הכרחי.

## בדיקה
בצע הזמנת בדיקה באתר (אפשר עם ספר אמיתי וסכום קטן) וודא שהמייל מגיע לתיבה. אם לא הגיע — בדוק בתיקיית הספאם, וודא שה-Access Key הודבק נכון בניהול.
